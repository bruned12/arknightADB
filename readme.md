# 介绍  
这是一个明日方舟全关卡自动刷图脚本  
和大部分录制操作不同，此脚本使用opencv模板匹配进行自动操作  
理论全关卡适配  
已实现功能  
1.自动刷图  
2.自动使用理智药  
# 使用方法
开启mumu模拟器的ADB功能  
直接clone整个储存库安装依赖即可未来会有打包好的exe版本
